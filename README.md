# Análise de Anúncios de Venda de Carros

## Descrição
Aplicação web interativa para análise exploratória de dados de anúncios de venda de veículos nos EUA. O projeto permite visualizar a distribuição de quilometragem e a relação entre quilometragem e preço dos veículos anunciados.

## Tecnologias utilizadas
- Python
- Pandas
- Plotly Express
- Streamlit
